# Smsaero

CLI client and API wrapper for SMSAero.ru

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'smsaero'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install smsaero

## Usage

    $ smsaero auth -l mylogin@example.com -p mystr0ngp@ss
    $ smsaero balance
    724.45
